💡 基本使用 💡

### 训练
rasa train --domain domain

### 多线程训练   
rasa train --domain domain --num-threads 12

### 开启 action 服务器   
rasa run actions   
rasa run -p 5011 --enable-api --cors "*" 

### 使用
通过命令行界面启动 action 服务器后，即可按照以下步骤进行对话
#### 1.使用网页对话
